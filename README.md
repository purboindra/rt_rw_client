# RT RW Apps Client

RT RW Apps Client is a Kotlin Multiplatform / Compose Multiplatform mobile application for community operations management. It connects to the RT RW backend API and supports resident authentication, announcements, activities, incident reports, notifications, and local data persistence.

This project demonstrates Kotlin Multiplatform mobile development using Compose Multiplatform, Ktor, Koin, Room, DataStore, Firebase Cloud Messaging, and a custom Express.js backend.

Backend: https://github.com/purboindra/rt_rw_backend
